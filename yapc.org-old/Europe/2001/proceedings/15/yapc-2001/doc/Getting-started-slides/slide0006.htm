OK, well done! You are now familiar with the base process of PerlPoint working. But there are several more things to discover, including tags, image inclusion, source nesting and on the fly source generation. And there are several more formats to convert sources into, currently including LaTeX, SDF, Clinton Pierce's Perl Projector. Further processing provides formatted text, PostScript and PDF. Read on in the tutorial which will explain these features in detail.
