6.1 pp2html

The Perl program pp2html is until now the most enhanced tranlator program for the PerlPoint format. pp2html generates a set of HTML files. Dependent on a variety of options and template files, the results can look quite different. Different colors, background images, frame sets, different text sizes or a Java Applet for the table of contents are layout elements which can be used to create a variety of documents. The following example shows a page from a Perl course, which has been written completely in PerlPoint.

Text Formatting: 


\B< .... >        Bold
\I< .... >        Italic
\C< .... >        Code
\SUB< ... >       Subscript
\SUP< ... >       Superscript
\U< .... >        Underlined
\E< html escape > z.B. \E<uuml>

Example code and verbatime blocks are presented in a colored boxes in pp2html. This is achieved by HTML tables with colored cells. The color of the cells and the text color in these cells can be individually influenced by the \BOXCOLORS tag: 


 \BOXCOLORS{bg=yellow fg=blue}

The default settings for background and foreground color are grey and black. The colors can be reset to their default values with 


 \BOXCOLORS{set=default}

Anchors, Hyperlinks, Images:

To use internal hyperlinks and cross references, you must insert so called anchors or link targets in your text. This is achieved by the \A tag. 


\A{name="name"}

For each page header such an anchor is automatically inserted, to make things easy. The names of these page anchors are simply the headers of the pages: 


\PAGEREF{name="page header"}     internal hyperlink with page number as text

\SECTIONREF{name="page header"}  internal hyperlink with page header as text
                                 \A{name="name"> is an anchor on the
                                 corresponding page

Hyperlinks to external pages can be defined with the following tag: 


\L{url="http://....."}<link text>

Images are included with the \IMAGE tag: 


\IMAGE{alt="Cool picture of xzy" align="left" src="xyz.gif"}

\IMAGE{src="xyz.gif"}      default for alt is "Image xyz.gif"
                           (with HTML as target language; with other target
                           languages the alt parameter may have no effect)

Note that the pathnames for the image files must be relative to the PerlPoint input file which contains the \IMAGE tag or they must be specified absolutely.

Cross References and Generation of Index:

Cross references are created with the \XREF tag: 


  \XREF{name=target}<text for cross reference>

For index entries there is the following tag: 


\X< ... >                    words are included in index and in output text
\X{mode="index_only"}< ... > words are included in index only

The index is created automatically at the end of the document unless the --no_index option of pp2html has been used.

Other Tags:

Line breaks and the known horizontal lines in HTML can be achieved be the following tags: 


   \LINE_BREAK     alias \BR
   \HR

Beside the tags, there are many options in pp2html which allow to control the appearance of the output. It is useful to write those options in options files (control files) which can be specified on the command line while starting the pp2html program: 


 pp2html @pp2html.cfg perlpoint.pp

In this example, all relevant options are contained in the file pp2html.cfg. More information about the options of pp2html can be seen when calling the program with the --help switch.
